# Thank you for thinking on how to contribute to this project!

## Ways you can contribute:
1. Raising an issue.
2. Improving the documentation.
3. Helping other community members by answering questions.
4. Making code more readable.
5. Writing fresh code for the project.
6. Fixing existing code, by closing issues.

## How to contribute: 
Usually the procedure goes as follows: 
1. Create a fork of the project.
2. Write your code in it.
3. Create a pull request.
4. If it closes an issue write: Closes [issue-number]# as title.

NOTE: If it is a good-first-issue or a simple documentation, then you can contribute without making a fork.
